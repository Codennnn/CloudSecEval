# 支持

Nest 是一个采用 MIT 许可证的开源项目，其持续发展离不开社区的支持。这个框架的诞生，源自一段漫长的旅程，包含了无数不眠之夜、下班后的坚持，以及忙碌的周末时光。

## 你可以如何支持？

Nest 没有大型公司持续投入资金来支持开发工作。我完全依赖于大家的善意 ❤️。不过，我也希望让这个框架变得更加强大，能够全身心投入，为你们带来让编码过程更愉快的优秀解决方案。为此，我运营了几个支持平台：

- 成为 [OpenCollective](https://opencollective.com/nest) 的捐助者或赞助者
- 通过 [PayPal](https://paypal.me/kamilmysliwiec) 进行一次性捐赠
- 或者直接联系我：[mail@kamilmysliwiec.com](mailto:mail@kamilmysliwiec.com)

如果你喜欢 Nest，或者你的企业正在使用 Nest，欢迎考虑赞助其开发，以确保你所依赖的项目能够**持续维护**和不断完善。同时，你的支持也将帮助我投入更多时间，为整个 Nest 社区创作有价值的内容，比如撰写教育类博客文章或录制教学视频。
