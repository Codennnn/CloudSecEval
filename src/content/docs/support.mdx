# 支持

Nest 是一个采用 MIT 许可证的开源项目，其持续发展离不开社区的支持。这个框架的诞生，源于一段漫长的旅程，凝聚了无数不眠之夜、下班后的坚持，以及周末的辛勤付出。

## 你可以如何支持？

Nest 并没有大型公司的持续资金投入，项目的推进完全依赖于社区每一位成员的善意支持 ❤️。我也希望让这个框架变得更加强大，能够全身心投入，为大家带来让编码过程更愉快的优质解决方案。为此，我运营了多个支持平台：

- 成为 [OpenCollective](https://opencollective.com/nest) 的捐助者或赞助者
- 通过 [PayPal](https://paypal.me/kamilmysliwiec) 进行一次性捐赠
- 或直接联系我：[mail@kamilmysliwiec.com](mailto:mail@kamilmysliwiec.com)

如果你喜欢 Nest，或者你的企业正在使用 Nest，欢迎考虑赞助项目开发，以确保你所依赖的框架能够持续维护并不断完善。同时，你的支持也将帮助我投入更多时间，为整个 Nest 社区创作有价值的内容，例如撰写技术博客或录制教学视频。
