# 拦截器

[常规拦截器](/interceptors)与 WebSocket 拦截器之间没有区别。下面的示例演示了一个手动实例化的方法作用域拦截器。与基于 HTTP 的应用程序类似，你也可以使用网关作用域拦截器（即在网关类前添加 `@UseInterceptors()` 装饰器）。

```ts
@UseInterceptors(new TransformInterceptor())
@SubscribeMessage('events')
handleEvent(client: Client, data: unknown): WsResponse<unknown> {
  const event = 'events';
  return { event, data };
}
```
