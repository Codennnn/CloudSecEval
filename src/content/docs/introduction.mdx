# NestJS 介绍

NestJS 是一个专为构建高效且可扩展的 [Node.js](https://nodejs.org/) 服务端应用而设计的框架。它采用现代 JavaScript 技术开发，完全基于 [TypeScript](http://www.typescriptlang.org/)，并对 TypeScript 提供完整支持，同时也允许开发者使用纯 JavaScript 编写代码。

NestJS 巧妙融合了面向对象编程（OOP）、函数式编程（FP）和函数响应式编程（FRP）等多种范式，为开发者带来灵活且强大的编程体验。

在底层实现上，Nest 基于强大的 HTTP 服务器框架构建。默认情况下使用 [Express](https://expressjs.com/) 作为底层框架，同时也支持配置为 [Fastify](https://github.com/fastify/fastify)。

Nest 在这些成熟的 Node.js 框架（Express/Fastify）之上提供了一层抽象封装，并直接开放底层框架的 API。这种设计让开发者既能享受 Nest 的架构优势，又能充分利用底层平台丰富的第三方模块生态。

## 设计理念

近年来，随着 Node.js 的普及，JavaScript 已成为前后端通用的开发语言。这一趋势催生了众多优秀项目，例如 [Angular](https://angular.dev/)、[React](https://github.com/facebook/react) 和 [Vue](https://github.com/vuejs/vue)。这些框架极大提升了开发效率，让前端应用变得更快、更易测试，也更易扩展。

然而，尽管 Node.js（以及服务端 JavaScript）生态中涌现了大量优质的库、工具和辅助模块，但在**应用架构**这一核心问题上，现有解决方案仍显不足。

Nest 正是为了解决这一痛点而诞生。它提供开箱即用的应用架构，帮助开发者和团队构建高度可测试、可扩展、低耦合且易于维护的应用。Nest 的架构设计深受 Angular 启发，将其成熟的设计理念引入 Node.js 服务端开发。

## 快速安装与项目创建

你可以通过两种主要方式开始使用 NestJS：一是使用 [Nest CLI](/cli/overview) 脚手架工具快速创建项目，二是[克隆现有的起始项目](#其他项目初始化方式)。这两种方式都能帮助你快速搭建项目的初始结构和基础配置，无论选择哪种方式，都能让你轻松起步。

使用 Nest CLI 创建新项目（推荐方式）：

```bash
$ npm i -g @nestjs/cli
$ nest new project-name
```

上述命令会创建一个新的项目目录，并自动生成核心 Nest 文件及相关模块，形成标准的项目基础结构。**首次使用 NestJS 的开发者推荐采用此方式**。我们将在[快速上手](/first-steps)章节详细介绍这种创建方式。

<CalloutInfo>
  如果希望创建更严格的 TypeScript 项目，可在 `nest new` 命令中添加 `--strict` 参数。
</CalloutInfo>

## 其他项目初始化方式

你也可以通过 Git 克隆 TypeScript 起始项目：

```bash
$ git clone https://github.com/nestjs/typescript-starter.git project
$ cd project
$ npm install
$ npm run start
```

<CalloutInfo>
  如果你希望克隆仓库但不保留 Git 历史记录，可以使用 [degit](https://github.com/Rich-Harris/degit)
  工具。
</CalloutInfo>

启动成功后，打开浏览器访问 `http://localhost:3000/`。

如需使用 JavaScript 版本的起始项目，只需将上述命令中的 `typescript-starter.git` 替换为 `javascript-starter.git` 即可。

你也可以手动安装核心依赖包，从零开始搭建项目。这种方式需要自行创建项目基础文件，并至少安装以下依赖：`@nestjs/core`、`@nestjs/common`、`rxjs` 和 `reflect-metadata`。如果你对此感兴趣，可以参考这篇实用教程：[5 steps to create a bare minimum NestJS app from scratch!](https://dev.to/micalevisk/5-steps-to-create-a-bare-minimum-nestjs-app-from-scratch-5c3b)。
