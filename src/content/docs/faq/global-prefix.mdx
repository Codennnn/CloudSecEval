# 全局前缀

若要为应用中的**所有路由**统一设置前缀，可以调用 `INestApplication` 实例的 `setGlobalPrefix()` 方法。

```ts
const app = await NestFactory.create(AppModule)
app.setGlobalPrefix('v1')
```

你也可以从全局前缀中排除特定路由：

```ts
app.setGlobalPrefix('v1', {
  exclude: [{ path: 'health', method: RequestMethod.GET }],
})
```

此外，你还可以直接以字符串形式指定要排除的路由路径，这会应用于该路径下的**所有**请求方法：

```ts
app.setGlobalPrefix('v1', { exclude: ['cats'] })
```

<CalloutInfo>
  `exclude` 选项中的 `path` 属性支持
  [path-to-regexp](https://github.com/pillarjs/path-to-regexp#parameters)
  包提供的通配符参数。注意：该配置不支持使用通配符星号 (`*`)，而必须使用路径参数（例如
  `:id`）或命名通配符（`*splat`）。
</CalloutInfo>
