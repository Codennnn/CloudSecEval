# HTTPS

要创建一个使用 HTTPS 协议的应用程序，需要在 `NestFactory` 类的 `create()` 方法中，通过 options 对象设置 `httpsOptions` 属性：

```ts
const httpsOptions = {
  key: fs.readFileSync('./secrets/private-key.pem'),
  cert: fs.readFileSync('./secrets/public-certificate.pem'),
}
const app = await NestFactory.create(AppModule, {
  httpsOptions,
})
await app.listen(process.env.PORT ?? 3000)
```

如果你使用的是 `FastifyAdapter`，可以按照如下方式创建应用：

```ts
const app = await NestFactory.create<NestFastifyApplication>(
  AppModule,
  new FastifyAdapter({ https: httpsOptions })
)
```

## 同时监听多个端口

下面的示例展示了如何实例化一个 Nest 应用，使其能够同时监听多个端口（例如同时监听一个非 HTTPS 端口和一个 HTTPS 端口）。

```ts
const httpsOptions = {
  key: fs.readFileSync('./secrets/private-key.pem'),
  cert: fs.readFileSync('./secrets/public-certificate.pem'),
}

const server = express()
const app = await NestFactory.create(AppModule, new ExpressAdapter(server))
await app.init()

const httpServer = http.createServer(server).listen(3000)
const httpsServer = https.createServer(httpsOptions, server).listen(443)
```

由于我们手动调用了 `http.createServer` / `https.createServer`，因此在调用 `app.close` 或收到终止信号时，NestJS 不会自动关闭这些服务器。我们需要自行处理关闭逻辑：

```ts
@Injectable()
export class ShutdownObserver implements OnApplicationShutdown {
  private httpServers: http.Server[] = []

  public addHttpServer(server: http.Server): void {
    this.httpServers.push(server)
  }

  public async onApplicationShutdown(): Promise<void> {
    await Promise.all(
      this.httpServers.map(
        (server) =>
          new Promise((resolve, reject) => {
            server.close((error) => {
              if (error) {
                reject(error)
              } else {
                resolve(null)
              }
            })
          })
      )
    )
  }
}

const shutdownObserver = app.get(ShutdownObserver)
shutdownObserver.addHttpServer(httpServer)
shutdownObserver.addHttpServer(httpsServer)
```

<CalloutInfo>
  `ExpressAdapter` 需要从 `@nestjs/platform-express` 包中导入。`http` 和 `https` 都是 Node.js
  的内置包。
</CalloutInfo>

<CalloutInfo type="warning">

此方案不适用于 [GraphQL 订阅](/graphql/subscriptions)。

</CalloutInfo>
